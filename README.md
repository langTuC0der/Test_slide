## Session 02 — Khái niệm tương tác

- **Hover** từ đỏ: tooltip định nghĩa ngắn  
- **Click**: panel giải thích đầy đủ  
- **Esc**: đóng panel  

## Điều khiển slide

| Phím | Hành động |
|------|-----------|
| `→` / `Space` / `Enter` | Slide tiếp |
| `←` | Slide trước |
| `Home` / `End` | Đầu / cuối |
| `Esc` | Đóng panel khái niệm |
